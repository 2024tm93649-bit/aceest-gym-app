# ACEest Gym DevOps Project

## Overview

Flask web application for gym management with CI/CD pipeline.

## Features

* Flask app
* Pytest testing
* GitHub Actions pipeline

## Run Locally

pip install -r requirements.txt
python app.py

## Run Tests

python -m pytest

## CI/CD

GitHub Actions automatically runs tests on every push.

## Jenkins

Jenkins is used as a build server for CI process.
